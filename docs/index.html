Worldseed Development Log

A concise record of the current vertical slice: what exists, what it feels like, and what remains intentionally out of scope.

Phase 1 Status

Worldseed now boots a local universe shard, persists it to SQLite, runs a deterministic simulation loop, and exposes the live archive through an SSH-backed terminal dashboard.

Features

Universe Shards

One SQLite database per universe, with deterministic creation from a persisted seed.

Append-Only Timeline

Events carry valid universe time and recorded archive time, keeping projections rebuildable later.

Remote Archive Daemon

worldseedd serves a Bubble Tea dashboard over a localhost SSH PTY session.

Living Simulation

Entropy drift, archive integrity, civilisation pressure, mutation, fragmentation, and collapse evolve while clients are disconnected.

Operator Intervention

preserve_archive improves integrity now and later records an unintended doctrinal consequence.

Quiet Phase Boundary

Federation is a placeholder only. No web app, multiplayer, ECS, or speculative distributed system is included.

Run Locally

worldseed --data-dir ./testdata universe create aurora
worldseedd --data-dir ./testdata start --universe aurora
worldseed connect
